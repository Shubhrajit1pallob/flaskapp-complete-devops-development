    • Expression and Functions (Terraform)

    This project demonstrates the use of expressions and functions in Terraform to manipulate and transform data structures such as lists and maps. It covers various operators, for expressions, splat expressions, and advanced techniques like transforming maps into lists and handling duplicate keys using ellipses (...).

    Data Sources (Terraform)

    This project demonstrates how to use Terraform data sources to fetch information from external sources dynamically. Data sources allow you to query existing resources or infrastructure details and use them in your Terraform configurations. This project covers fetching AMI IDs, AWS account information, VPC details, availability zones, and creating IAM policy documents.

    S3 Static Website

    This project demonstrates how to deploy a static website hosted on Amazon S3 using Terraform. The setup includes creating an S3 bucket, configuring it for public access, applying a bucket policy to restrict write access, and uploading the website files (index.html and error.html) to the bucket.

    VPC and EC2 (Terraform)

    This project demonstrates how to deploy a Virtual Private Cloud (VPC) and an EC2 instance using Terraform. The setup includes creating essential networking components, configuring security groups, and launching an EC2 instance with a publicly available NGINX AMI.
